Proyecto Web + Ejercicios de Condicionales

Consigna general

Consigna: Diseñar un sitio web completo que incluya:

  • Un archivo .html con los elementos necesarios para mostrar tu temática asignada (estructura semántica, secciones, navegación).
  • Un archivo .css con los estilos para darle diseño y presentación a tu sitio.
  • Un archivo .js con las actividades de alternativa condicional, usando prompt() para pedir datos y alert() o console.log() para mostrar los resultados.
  • Todos los archivos deben tener nombres significativos y estar organizados en una misma carpeta.

Objetivos:

  • Practicar el uso de etiquetas HTML, CSS y condicionales en JavaScript.
  • Relacionar la programación con tu temática específica.
  • Desarrollar hábitos de orden y buena organización de archivos.
  • Fomentar la creatividad y el diseño web.
  • Mejorar la capacidad de resolver problemas mediante condicionales.
